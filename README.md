# parseium-skill

Agent skill for extracting structured data from social media and web platforms via [Parseium](https://parseium.com) APIs.

## Supported Platforms

- Instagram (profiles, posts, reels)
- TikTok (profiles, videos, comments)
- YouTube (channels, videos, transcripts, comments)
- Reddit (posts, users)
- LinkedIn (profiles, companies, posts)
- Zillow (search, property details)
- Indeed (job postings)
- App Store (apps, search)

## Requirements

- API key from [parseium.com/api-key](https://www.parseium.com/api-key)

## Installation

Add to your agent's skills directory or reference via skill registry.

## Usage

Once installed, ask your agent to fetch data from supported platforms:

- "Get NASA's Instagram profile"
- "Extract transcript from this YouTube video"
- "Search Zillow for homes in Austin TX"
